# Knowledge Capture

After resolving a difficult bug or implementing a complex feature:

1. **Identify**: Did we solve something undocumented? Invent a repeatable pattern?
2. **Check**: List `.cursor/skills` for existing skills to update.
3. **Update/Create**: Edit SKILL.md or create new skill under `.cursor/skills/<name>/`.
4. **Workflows**: If the solution involved a sequence, create `.cursor/rules/<name>.mdc` or `.cursor/commands/<name>.md`.
5. **Document**: Update task.md or walkthrough.md with changes.
